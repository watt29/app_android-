# GitHub JSON สำหรับอุ่นใจ

ไฟล์นี้ใช้เป็นข้อมูลกลางให้แอปดึงคำสั่ง รายชื่อ และเบอร์เสี่ยงจาก GitHub แบบอัปเดตได้ทันที

## วิธีใช้แบบง่าย

1. เปิดไฟล์ [`github-bundle.example.json`](./github-bundle.example.json)
2. แก้ข้อมูลในไฟล์ให้เป็นของคุณ
3. อัปโหลดไฟล์นี้ไปไว้ใน GitHub repository
4. เปิดไฟล์แบบ `Raw`
5. เอาลิงก์ Raw มาใส่ในช่อง `ตั้งค่า GitHub JSON` ในหน้าแอดมินของแอป

## โครงข้อมูลที่แอปรับ

- `botCommands` สำหรับคำสั่งบอท
- `contacts` สำหรับรายชื่อ
- `scamNumbers` สำหรับเบอร์มิจฉาชีพ
- `scamSources` สำหรับแหล่งอ้างอิง

## ตัวอย่างลิงก์ Raw

```text
https://raw.githubusercontent.com/username/repo/main/github-bundle.json
```

## หมายเหตุ

- ถ้าใส่ลิงก์แล้วกดบันทึก แอปจะดึงข้อมูลให้ทันที
- ถ้าเน็ตมีปัญหา แอปจะใช้ข้อมูลที่เคยเก็บไว้ในเครื่องก่อน
